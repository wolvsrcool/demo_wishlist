# To run the application locally

## 1. Clone the repository

```bash
git clone https://github.com/wolvsrcool/demo_wishlist.git
```

## 2. Install dependencies

```bash
   cd demo_wishlist
   npm install
```

## 3. Running the App

### Development Mode

Starts the Vite dev server with hot reload:

```bash
npm run dev
```

### Production Preview

Build the project and preview the production build:

```bash
npm run build
npm run preview
```

## 4. Start the JSON Server (API)

The app expects a local JSON server to serve API data.<br>
In a separate terminal:

```bash
npm run server
```
